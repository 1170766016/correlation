# 🚀 项目启动与操作指南

本工具专门用于分析终检 Fail 产品的全流程共性聚集度（Lift 提升度分析）。

## 1. 如何启动分析平台

在项目根目录下打开终端，运行以下命令：

```bash
# 使用内置虚拟环境启动 Streamlit 应用
.conda\Scripts\streamlit.exe run app.py
```

启动后，在浏览器访问：`http://localhost:8501`

---

## 2. 大规模数据预处理 (可选)

由于 Excel (.xlsx) 解析较慢，处理 10 万行以上数据时，建议先将其转换为 CSV 格式。

**操作命令：**
```bash
# 转换单个文件
.conda\python.exe excel_to_csv_converter.py "你的数据.xlsx"

# 批量转换当前目录下所有 Excel
.conda\python.exe excel_to_csv_converter.py .
```

---

## 3. 性能优化与配置

*   **修改 API 地址**：在 `app.py` 顶部的 `LLM_API_URL` 处修改。
*   **修改分析列名**：如果您需要调整分析的 200 个列名，请参考 `性能优化与列配置指南.md`。

---

## 4. 故障排查

*   **端口占用**：如果提示 8501 端口被占用，可以改用其他端口启动：
    `streamlit run app.py --server.port 8888`
*   **缺少列名**：如果上传文件后报错，请检查您的 Excel 表头是否包含 `Results`、`Date`、`Failed_Station`、`Failure_Mode` 这几个核心列。
