

## 1. 配置列名白名单 (Focus Columns)

在 `app.py` 的 `CONSTANTS` 区域（约第 25 行之后）添加您的目标列列表：

```python
# 您关心的 200 个列名（示例）
FOCUS_COLS = [
    'Project', 'Build', 'Config', 'Aging_Mode', 'Serial_No',
    'VCM_M1_FPC_Load_End_Time', 'VCM_M1_FPC_UpCoil_Attach_MC_ID',
    # ... 在这里粘贴完整的 200 个列名 ...
]

# 系统必须加载的基础列
REQUIRED_COLS = ['Results', 'Date', 'Failed_Station', 'Failure_Mode']

# 最终加载列表
LOAD_COLS = list(set(FOCUS_COLS + REQUIRED_COLS))
```

## 2. 开启“按需加载” (usecols) 优化

在 `app.py` 的 `main()` 函数中，找到读取文件的代码块（约第 315-325 行），修改为：

```python
if uploaded:
    try:
        if uploaded.name.endswith('.csv'):
            # 优化：仅加载指定的列，大幅减少内存占用
            df_raw = pd.read_csv(uploaded, usecols=LOAD_COLS)
        else:
            # 优化：Excel 同样支持 usecols
            df_raw = pd.read_excel(uploaded, engine='openpyxl', usecols=LOAD_COLS)
            
        # 优化：类型压缩，将字符串转为分类类型
        for col in df_raw.select_dtypes(include=['object']).columns:
            if col not in REQUIRED_COLS: # 核心列保持原始类型
                df_raw[col] = df_raw[col].astype('category')
                
    except Exception as e:
        st.error(f"加载失败，请检查列名是否存在: {e}")
```

## 3. 性能建议总结

1.  **优先使用 CSV**：40 万行数据读取，CSV 只需几秒，Excel 可能需要几分钟。
2.  **提前筛选**：如果只需分析 `Failed_Station == 'METROLOGY'`，可以在读取后立即执行 `df_raw = df_raw[df_raw['Failed_Station'] == 'METROLOGY']` 并释放原内存。
3.  **内存监控**：在 Streamlit 中，可以通过 `st.write(f"当前内存占用: {df_raw.memory_usage().sum() / 1e6:.2f} MB")` 监控数据大小。
